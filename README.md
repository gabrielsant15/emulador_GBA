# 🎮 Emulador GBA Web

Um emulador de Game Boy Advance funcional que roda direto no navegador. Desenvolvido para ser leve, rápido e compatível com dispositivos móveis e desktop.

![Status](https://img.shields.io/badge/Status-Online-brightgreen)
![Platform](https://img.shields.io/badge/Platform-Web-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

## 🚀 Como Jogar

1. Acesse o link do projeto via GitHub Pages.
2. Clique no botão **"CARREGAR JOGO (.GBA)"**.
3. Selecione o arquivo da sua ROM (ex: `Pokemon.gba`) no seu computador ou celular.
4. O sistema irá carregar o motor e iniciar a dungeon automaticamente!

## ⌨️ Controles (Teclado)

| Botão GBA | Tecla PC |
| :--- | :--- |
| **D-Pad (Setas)** | Setas do Teclado |
| **Botão A** | Z |
| **Botão B** | X |
| **Botão L** | Q |
| **Botão R** | E |
| **Start** | Enter |
| **Select** | Shift |

> **Dica:** Se o som não iniciar de primeira, clique em qualquer lugar na tela do jogo para ativar o áudio no navegador.

## 🛠️ Tecnologias Utilizadas

* **HTML5 / CSS3** (Interface customizada)
* **JavaScript** (Lógica de carregamento de arquivos)
* **EmulatorJS** (Núcleo de emulação em WebAssembly)
* **GitHub Pages** (Hospedagem)

## 💾 Sistema de Save
O emulador suporta salvamento interno. Para baixar seu progresso ou carregar um save antigo, utilize o menu flutuante que aparece ao passar o mouse na parte inferior da tela do emulador.
